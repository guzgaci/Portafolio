 Portafolio Python & SQL

Este repositorio forma parte de mi portafolio técnico y complementa la información de mi CV. 
Aquí muestro ejercicios y mini–proyectos de **análisis de datos** utilizando **Python** y **SQL**.

## Objetivo

Analizar información de una empresa ficticia:
- Empleados y departamentos
- Salarios
- Cursos de capacitación (por ejemplo, TIC)

El enfoque es **limpio y didáctico**, para que cualquier reclutador pueda ver cómo estructuro código,
documento mis consultas y resuelvo problemas con datos.

## Estructura del repositorio

- `python/`
  - `02_estructuras_datos.py`: uso de listas, tuplas, diccionarios y manejo básico de datos.
  - `03_funciones_y_archivos.py`: funciones reutilizables y lectura/escritura de archivos.
  - `04_numpy_pandas.py`: análisis tabular con NumPy y pandas.

- `sql/`
  - `01_consultas_basicas.sql`: SELECT, filtros, ordenamiento.
  - `02_joins.sql`: combinación de tablas (INNER/LEFT JOIN).
  - `03_agregaciones_y_groupby.sql`: métricas agregadas por departamento, puesto, etc.
  - `04_subconsultas.sql`: consultas más avanzadas usando subconsultas.

## Tecnologías

- Python (listas, diccionarios, funciones, archivos, NumPy, pandas)
- SQL (consultas, joins, agregaciones, subconsultas)
- Git y GitHub para control de versiones

## Cómo usarlo

1. Clonar el repositorio.
2. Revisar primero los scripts de `python/`.
3. Ejecutar las consultas en `sql/` sobre una base de datos de ejemplo (por ejemplo MySQL o PostgreSQL).
